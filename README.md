# Totus Tuus Keely Portfolio Starter

This is a plain HTML/CSS portfolio designed for GitHub Pages. It does not require
React, Node, Ruby, Jekyll, or a local build process.

## Uploading it to GitHub

1. Open the `TotusKeely/totuskeely.github.io` repository.
2. Delete the old `index.md` file so there is only one homepage.
3. Upload everything in this folder to the repository root.
4. Commit the changes to the `main` branch.
5. Open the live site after GitHub Pages finishes publishing.

## Required edits before sharing the site

Search `index.html` for these placeholders:

- `YOUR_EMAIL@example.com`
- `YOUR_LINKEDIN_URL`

Then:

- Add your current résumé as `assets/Totus-Tuus-Keely-Resume.pdf`.
- Replace the four SVG placeholders in `assets/images/` with your own approved
  images. You can keep the same filenames to avoid changing the HTML.
- Verify every number and technical detail before publishing.
- Do not upload proprietary CAD, customer information, restricted drawings,
  confidential dimensions, internal pricing, or employer-owned source files.

## Best first images

1. Spray booth: one employer-approved overall render and one detail image.
2. Motor control: hardware photo plus command-versus-response plot.
3. Robotics: RoboDK screenshot or GIF plus coordinate-frame diagram.
4. Cart: finished cart photo plus wiring or system diagram.

## Editing

You can edit `index.html` and `styles.css` directly in GitHub:

- Open the file.
- Click the pencil icon.
- Make the change.
- Click **Commit changes**.

The site is intentionally kept simple so the engineering work remains the focus.
