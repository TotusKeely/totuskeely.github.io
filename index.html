Industrial Crossdraft Spray Booth

CASE STUDY 02

DC Motor Position Control

A closed-loop controls project using encoder feedback to command and measure the angular position of a geared DC motor.

System

  • Arduino Uno and Adafruit Motor Shield V2.
  • 100:1 gearmotor with encoder feedback.
  • Python/Tkinter interface for commands and measured response.
  • Proportional-derivative control with experimentally evaluated gains.

What to show

  • A labeled hardware photograph or wiring diagram.
  • The control block diagram and the governing equations.
  • A commanded-versus-measured angle plot.
  • How changing the gains affected rise time, overshoot, and settling.

CASE STUDY 03

6-DOF Robotic Pick-and-Place

A RoboDK simulation using a HIWIN industrial robot and RobotiQ gripper to complete a repeatable three-point pick-and-place operation.

Technical work

  • Defined robot, tool, and target coordinate frames.
  • Computed and checked 4×4 homogeneous transformation matrices.
  • Verified rotation-matrix orthogonality and determinant.
  • Developed motion logic and validated the simulated sequence.

What to show

  • A GIF of the complete motion cycle.
  • A frame diagram showing base, flange, tool, and target relationships.
  • A small, readable code excerpt for transform or motion logic.
  • The verification steps used to confirm the transforms were valid.

Replace with a cart photograph and a system diagram.

CASE STUDY 04

Autonomous Distance-Control Cart

An embedded project combining ultrasonic distance measurement, directional motor commands, and PWM speed control on a small mobile platform.

Technical work

  • Read distance using an HC-SR04 ultrasonic sensor.
  • Controlled left and right motors with separate PWM outputs.
  • Implemented threshold-based movement logic and tuned motor duty cycles.
  • Troubleshot CircuitPython libraries, board definitions, and USB communication.

What to show

  • A short demonstration video or GIF.
  • A wiring diagram and signal-flow diagram.
  • A concise flowchart of the decision logic.
  • The main challenge you encountered and how you resolved it.
